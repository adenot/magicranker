About The Site

Magic Ranker is an ASX (Australian Stock Exchange) implementation of the Magic Formula ranking system from Joel Greenblatt's book, The Little Book That Beats The Market. The book is mandatory reading before using this tool.

The data is provided by Yahoo Finance and should only be used for personal use. I can't vouch for the integrity of the data.

Magic Ranker is a not-for-profit site. I don't make any money off it, nor do I intend to. If bandwidth costs pick up, I may request donations to keep the site running. The site is here because I really liked the ideas in Joel's book but couldn't find an online tool like MagicFormula.com for the ASX.

The site was written in Python using the Django framework.

How It Works

Data is scraped from the Yahoo Finance on a daily basis and a ranking script runs that:

  1. Sorts the Return on Equity values collected from highest to lowest and assigns an rank from 1 onward
  2. Sorts the P/E fields from lowest to highest and assigns a rank from 1 onwards.
  3. Adds the ROE and PE rank together to get a total rank and then orders by the total rank.
In addition, my application differs from the Magic Formula in that it also offers the ability to average the ROE by either 5 years or 10 years, ( if companies data doesn't go back that far, they're eliminated) and the ROE in step 1 is replaced with an average ROE. I thought this was a nifty feature to have but it's not in the book, so feel free to ignore it.
