About The Site

Magic Ranker is an ASX (Australian Stock Exchange) ranking tool based on the Magic Formula ranking system from Joel Greenblatt's book, The Little Book That Beats The Market.

Magic Ranker is a not-for-profit site. The site exists because I really liked the ideas in Joel's book but couldn't find an online tool like MagicFormula.com for the ASX.

The site was written in Python using the Django framework. The ranking and averaging is handled by the incredible pandas library. The source code is available on Github.

Disclaimer

The data is sourced from various places on the internet and should only be used for personal use.
